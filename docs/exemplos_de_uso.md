# Exemplos de Uso

## Caso 1: Quero verificar se o processo do cron está rodando
1. Ir até o diretório com o script `verifica_processo.sh`
```bash
cd scripts/sistema
```
2. Atualizar as permissões para conseguir rodar o script
```bash
chmod +x verifica_processo.sh
```
3. Rodar o script
```bash
./verifica_processo.sh
```
4. Dar o nome do processo que deseja verificar (cron)
```bash
> Informe o nome do processo que deseja verificar: cron
```
